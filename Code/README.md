# Code repository
### Questa pagina contiene il codice creato per la tesi "PATIENT KNOWLEDGE GRAPH REPRESENTATION USING NON-EUCLIDEAN METRIC"
## Gli studi fatti in ordine sono:
#### 1. Studi preliminari su node embedding con Node2Vect
#### 2. Esempi di rappresentazione iperbolica tramite Poincaré Disk
#### 3. Comparazione dei modelli GCN e HGCN su dataset medici noti al fine di identificare gli iperparametri più influenti
#### 4. Applicazione dei modelli GCN e HGCN al caso studio delle malattie medeliane
#### 5. Applicazione dei modelli AttE e AttH al caso studio tramite Knowledge Graph
#### 6. Analisi impatto dei dati prelevati dal Knowledge Graph del caso studio per valutarne la coerenza biomedica
#### 7. Confronto finale con analisi impatto delle varie scelte
